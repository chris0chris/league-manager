# Frontend Template

A standardized template for creating new React-based frontend applications within LeagueSphere.

## Features
- **Build Tool**: Vite for fast development and bundling.
- **Language**: TypeScript for type safety.
- **Styling**: Vanilla CSS (standard across the project).
- **Standards**: Pre-configured ESLint and Vitest.

## Usage
When creating a new frontend application, use this directory as a base to ensure consistency with project standards and integration with the Django backend.
