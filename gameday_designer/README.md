# Gameday Designer

A sophisticated React-based flowchart tool for creating, managing, and applying tournament schedule templates.

## Architecture
- **Hierarchy**: Field > Stage > Game.
- **Engine**: Uses React Flow for visual representation and wiring.
- **Progression**: Advanced logic for winner/loser paths in tournament brackets.

## Development
- **Location**: `gameday_designer/src/`
- **Tests**: `npm run test:run`
- **Build**: `npm run build`

Refer to `gameday_designer/GEMINI.md` for detailed agent-specific instructions.
