# Officials App

Manages referee (officials) information, licensing, and assignments to games.

## Features
- Database of certified officials with license levels.
- Automated and manual assignment of officials to games.
- Reporting of official duties for reimbursement or tracking.

## Components
- `api/`: Endpoints for official-related data.
- `forms.py`: Management interfaces for official rosters.
- `views.py`: Logic for official dashboards and assignment views.
