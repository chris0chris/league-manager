Export dropdown contains: "Export Schedule" + "Export Template" — template save is hidden here.
Generate Tournament opens a modal that also lets users pick a tournament format (a kind of template).
No UI for browsing, loading, or managing saved server-side templates.

The Problem

Cramped

Template actions (save/load) buried in export dropdown and tournament generator modal — no clear home.

Missing

The backend has full template CRUD (list, create, delete, clone, preview, validate) but no UI exists to use it.

What needs a better home?

  • 📁 Browse templates — list all saved templates for the association
  • 💾 Save as template — save current schedule as a reusable template
  • ▶️ Apply template — load a template onto current gameday
  • 🏆 Generate tournament — pick a tournament format and auto-build a schedule
  • 📤 Export / Import — JSON file import/export
  • 🔧 Manage templates — rename, delete, clone, preview templates
