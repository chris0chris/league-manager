# LeagueSphere Demo Environment

**URL:** https://demo.leaguesphere.app

Welcome to the LeagueSphere demo! This is a fully functional demo of the LeagueSphere platform where you can explore all features with predefined accounts and realistic sample data.

## Quick Start

### Demo Accounts

Four demo accounts are available with different permission levels:

| Username | Password | Role | Access |
|----------|----------|------|--------|
| `admin@demo.local` | `DemoAdmin123!` | Administrator | Full system access, user management |
| `referee@demo.local` | `DemoRef123!` | Referee | Match officiating and scoring |
| `manager@demo.local` | `DemoMgr123!` | Team Manager | Team management and roster control |
| `user@demo.local` | `DemoUser123!` | Regular User | View standings and league information |

### Getting Started

1. Navigate to https://demo.leaguesphere.app
2. Click "Login" and enter any demo account credentials
3. Explore the system features
4. All functionality works exactly as in production

## Important Notes

### Daily Reset

The demo database automatically resets every day at **midnight UTC**. This means:
- All changes you make will be cleared
- Demo accounts are restored
- Sample data is refreshed to original state
- Reset typically completes in under 10 seconds

### Data Privacy

- All data in the demo is completely synthetic
- No real personal information is stored
- No connection to production systems
- Safe to enter any test data you want

### Demo Limitations

- External integrations (payments, third-party APIs) are disabled
- Email functionality uses console output only
- Some admin features may be restricted for demo safety
