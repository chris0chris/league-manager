# Documentation Hub

Central repository for all project documentation, organized by category.

## Structure
- `arch/`: Architectural Decision Records (ADR) and system design documents.
- `features/`: Documentation for specific LeagueSphere features (current and historical).
- `guides/`: Onboarding, setup, and contribution guides.
- `plans/`: Implementation roadmaps and project plans.
- `reports/`: Test results, verification reports, and performance analysis.
- `testing/`: Test strategies, scenarios, and QA protocols.

## Primary Resource
The **[Contributor Guide](guides/contributor-guide.md)** is the main entry point for developers and agents.
