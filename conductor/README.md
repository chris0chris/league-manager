# Conductor

Conductor is the project's internal planning and task tracking system. It manages "tracks" of work, specifications, and implementation plans.

## Structure
- `tracks/`: Active development tracks.
- `archive/`: Completed or cancelled tracks.
- `code_styleguides/`: Language-specific coding standards (Python, TypeScript, etc.).
- `product.md`: High-level product definition.
- `tech-stack.md`: Documentation of the technologies used in LeagueSphere.
- `workflow.md`: Internal development workflows and processes.

## Usage
Refer to `conductor/index.md` (if present) or `tracks.md` for an overview of current work.
