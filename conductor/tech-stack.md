# Tech Stack

## Backend
- **Django 5.2+**: Core web framework.
- **Django REST Framework (DRF)**: API layer.
- **Knox**: Token-based authentication.
- **MySQL/MariaDB**: Persistent storage.

## Frontend
- **React 19**: UI library.
- **Vite**: Build tool and dev server.
- **TypeScript**: Static typing.
- **React-Bootstrap**: Component library.
- **React Flow** (legacy) / **Custom Hierarchical State**: Designer logic.
- **i18next**: Internationalization.

## Testing & Quality
- **pytest**: Backend testing.
- **Vitest**: Frontend unit and integration testing.
- **ESLint**: JavaScript/TypeScript linting.
